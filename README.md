# Ecem Kemal - Interior Design Portfolio

Dieses Projekt ist eine moderne, minimalistische und dynamische Portfolio-Website für Innendesign. 
Entwickelt als Full-Stack-Anwendung mit Next.js 15 und Supabase.

## 🚀 Technologien

* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Datenbank & Storage:** [Supabase](https://supabase.com/)
* **Styling:** Tailwind CSS
* **Sprache:** TypeScript

## ✨ Features

* **Dynamisches Projektmanagement:** Neue Projekte können einfach über das Admin-Panel hinzugefügt werden.
* **Erweiterte Galerie:** Projektdetails mit einem eleganten 3-Spalten-Grid und Lightbox-Funktion (Bilder vergrößern).
* **Responsive Design:** Vollständig optimiert für alle Mobil- und Desktop-Geräte.
* **Moderne Architektur-Ästhetik:** Fokus auf minimalistische Typografie und abgerundete Designelemente.

## 🛠️ Installation

1.  Abhängigkeiten installieren:
    ```bash
    npm install
    ```
2.  Erstellen Sie eine `.env.local` Datei und fügen Sie Ihre Supabase-Zugangsdaten hinzu.
3.  Anwendung starten:
    ```bash
    npm run dev
    ```
